# 응용계층의 역할

> OSI 모델 중 응용계층의 역할



- 서비스를 요청하는 측 : 클라이언트
- 서비스를 제공하는 측 : 서버



"" 애플리케이션은 사용자 측면에서 보자면 서버, 사용자는 클라이언트 ""



#### 응용계층은  "세션계층 + 표현계층"



클라이언트 측과 서버 측 애플리케이션이 통신하려면 응용계층의 '프로토콜'을 이용

- 웹사이트 = 'HTTP'
- 파일전송 = 'FTP'
- 메일전송 = 'SMTP'
- 메일수신 = 'POP3'

또한 컴퓨터나 네트워크 장비에 붙여진 이름을 기반으로 IP 주소를 알아내는 것을 이름해석

# 무선 랜의 구조



## 1. 무선 랜이란 ?

- 랜 케이블이 없는 상태를 뜻함



#### 구성

- 무선 액세스 포인트 ( Wireless Access Point )
- 무선 클라이언트 ( 컴퓨터나 스마트폰 등 )



###### 컴퓨터의 구성품

- 무선 랜 칩
- 무선 랜 어댑터



## 2. 인프라스트럭처 방식과 애드혹 방식이란?

> 무선 랜을 연결하는 방식
>
> `인프라스트럭처, 애드혹`



- 인프라스트럭처 ( 주로사용 )
  - 무선 엑세스 포인트를 통하여 통신하는 방식

- 애드혹
  - 무선 클라이언트끼리 직접 통신하는 방식



## 3. 무선 랜 규격

> IEEE802.11

11n, 11ac 를 주로 사용
